# ReceiptsAT

**Emissão de recibos pelo portal da Autoridade Tributária (Portugal)**

A lightweight Windows desktop application for generating and managing rent receipts through Portugal's Tax Authority portal (Portal das Finanças).

## 🎯 Features

- ✅ Generate official rent receipts
- ✅ Secure login with Portal das Finanças
- ✅ Export receipts to CSV
- ✅ View receipt history
- ✅ No Python or external dependencies required
- ✅ Self-contained Windows executable

## 📋 Quick Start

### System Requirements

- **Operating System:** Windows 10 or Windows 11 (64-bit)
- **RAM:** Minimum 2GB
- **Disk Space:** ~100MB for installation
- **Internet Connection:** Required for Portal das Finanças access

## 🚀 Getting Started

1. Download the latest installer from [Releases](../../releases)
2. Run `PortalReceiptsApp_Setup_v*.exe`
3. Follow the setup wizard
4. Launch the application and log in with your Portal das Finanças credentials
